# Привет

OneScript.Web - это веб-сервер, основанный на технологии Microsoft ASP.NET и Kestrel, использующий среду исполнения OneScript в качестве прикладного серверного кода.

Если предыдущая фраза была непонятна, то: Это веб-приложения написанные на языке 1С.
